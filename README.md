# React + TypeScript + Vite

این پروژه فروشگاه لوازم دجیتال، شامل صفحه محصولات، درباره ما، و صفحه اصلی است. پروژه با تکنولوژی React و TypeScript و Vite ساخته شده است.

## نحوه نصب

npm install

## اجرای پروژه

npm run dev

## ساختار پوشه ها

src/
assets/
components/
data/
pages/
utils/

## توضیحات

- assets: پوشه ای است که فایل های Assets مانند تصاویر و فونت ها در آن قرار می گیرند.

- components: پوشه ای است که فایل های Component قرار می گیرند. Component ها اجزای کوچک و مجزا از UI هستند که می توانند در وبسایت استفاده شوند.

- data: پوشه ای است که فایل های Data قرار می گیرند. Data ها داده های مورد نیاز برای UI هستند.

- pages: پوشه ای است که فایل های Page قرار می گیرند. Page ها صفحات اصلی و اصلی وبسایت هستند.

- utils: پوشه ای است که فایل های Utility قرار می گیرند. Utility ها ابزارهای کوچک و کاربردی هستند که می توانند در پروژه استفاده شوند.
